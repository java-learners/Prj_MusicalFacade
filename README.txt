Musical Facade is an interactive installation for house projections

(c) Martin Froehlich (code) and Marco von Baumgartner (music) for http://projektil.ch

Its based on the Processing.org framework but uses some extended libraries in order to work
These libraries can be found inside the <libs> directories.

Inside directory <externals> are 

a maxMSP patch to configure the facade ("Schlosskonfigurator.maxmsp")
a pd patch for the music ("mlaf_4.pd")

all communications are based on OpenSoundControl
